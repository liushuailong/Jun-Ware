# Jun Ware

## 简介

钧瓷（Jun ware）：河南禹县一带生产的以铜为着色剂，釉色呈紫红、天蓝、月白等多种色调的陶瓷制品。  
本项目用来学习浏览器引擎的构建（使用rust语言），参考Matt Brubeck的项目`Let's build a browser engine!`来理解浏览器运行的原理。

## 目的

- 了解当今浏览器的种类，和所使用的引擎的发展历史。
- HTML rendering engine
